What is task-chrono?

task-chrono is a tool that helps in task planning. It does this by providing statistics on how accurately you estimate task duration. This will give insight in how good you are at estimating task duration, which in turn will help planning your work.

The applicaton menu shows the following options:

  • List: the list of the tasks in your current list
  • Stats: estimation statistics
  • Settings: configure the application behaviour and manage lists
  • Help: this page
  • Log out: log out from the application

Task list

The task list shows all the tasks in the current list, sorted by date and status. Tasks in progress are on top, followed by newly created tasks, and finally finished tasks at the bottom. In all three categories, the oldest tasks is on top.

Adding a new task

To add a new task to the list, simply type a task name in the left field at the top of the page. In the second field, enter an estimate of how long you think it will take to complete the task (see hereafter). Click on the Create button to add the task to the list.

Estimated duration

Entering an estimate for task duration, use the following format:

NNd NNh NNs

Here, NN is any number, and d denotes work days, h denotes hours, and m minutes. The order of the groups is not important, nor are all mandatory: one is enough. As an example, the following all signify a duration of 9 hours and 30 minutes (assuming that one work day is set to be 8 hours; see below):

     1d 1h 30m
     9h 30m
     1.125d 0.5h
     1.1875d
     9.5h
     570m

Starting and stopping a task

Starting, stopping, an restarting a task is done by simply clicking on task name in the list. Active running tasks are highlighted in green, with the duration being updated in real time. Finished tasks are shown in a light grey, but can be restarted by again clicking on the task name. New tasks can be recognized by having no duration logged.

Task properties

By clicking on the line below the task name, the task properties are reveiled. These properties are the creation date and time, the moment it was first started, the moment it was last started, and the moment it was finished.

Deleting a task

Below the tasks properties a button is shown to delete the task from the list. Removing a large number of old tasks can be done via the settings page.
